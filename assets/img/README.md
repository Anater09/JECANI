# Carpeta `assets/img/`

El `index.html` referencia las imágenes de esta carpeta usando **rutas
locales** (por ejemplo `assets/img/hero-bg.jpg`), en lugar de las URLs
externas de Google que traía el diseño original. Esto es una buena
práctica: así el sitio no depende de un servicio externo y las imágenes
cargan más rápido.
